如何在电视上看Netflix

第一步 有个不错的电视

想要在电视上观看Netflix，需要配备一部能够下载 Netfilx APP的装置，可以是目前市面是普遍售卖的智能电视Netflix推荐，或者是使用一些电视盒子来支持Netflix App。

第二步 部署宽带环境

观看Netflix 需要特殊的网络环境，这块市面上有层出不穷的方。对于技术困难或者想花钱了事的朋友，万能淘宝也许是个不错的选择。

第三步 安装Netflix APP

现在国内的电视自带的应用市场，一般可拓展性比较差，所以这时候推荐U盘安装的方式，只要在网上下载对应软件的APK放到自己的U盘，然后把U盘插到电视，在电视的应用助手或者文件管理等简单操作下即可安装。APK下载的地方推荐MirrorAPK，在搜索框中搜索Netflix TV（白底红字APP是TV版的），重复上述操作即可。

第四步 登陆Netflix

登陆注册通过网站，支付需要一张Visa/Mastecard/美国运通，大概HK95$(约73RMB)，在电视上登陆后就可以开始畅游Netflix！

ToB产品的三层Aha moment

User level Aha: 比如，一个gitlab用户完成basic setup之后提交了第一行代码。他解决了一个问题，明白了产品feature的价值。但是这只是开始。

Team-level Aha：通常通过协作的方式。比如一个两三个人的小team开始用免费版。

Organization-level Aha：team 体会到了产品价值以后，需要pitch到别的部门或者决策层。这时候需要让他们很清楚地了解paid version ROI是什么。

没有很好地设计好这三个aha moment, 就意味着sales需要在用户还对于付费还没有ready的时候就要介入，这就违背了PLG的本意。
